About NLPCraft

NLPCraft is an open source library for building Natural Language Interface for any type of applications.

NLPCraft is simple to use: define a semantic model and intents to interpret user input. Deploy this model and use REST API to explore the data using natural language from your applications.

Learn more

Why NLI

Natural Language Interface (NLI) enables users to explore any type of data sources using natural language augmenting existing UI/UX with fidelity and simplicity of conversational AI.

There is no learning curve, no special rules or applications to master, no syntax or terms to remember - just a natural language that your users already speak and the tools they already use.

Key Features
Semantic Modeling

Combination of advanced semantic modelling and comprehensive intent-based matching allows the unprecedented understanding of the natural language.

Any Data Source

Any data source, device, or service - public or private. From databases and SaaS systems, to smart home devices, voice assistants and chatbots.

Java-First

REST API and Java-based data modelling natively support world's largest ecosystem of development tools, programming languages and services.

Strong Security

HTTPs, model deployment isolation, 256-bit encryption and ingress-only connectivity are among the key security features in NLPCraft.

Model-As-A-Code

Model-as-a-code convention natively supports any system development life cycle tools and frameworks in Java eco-system.

Short-Term-Memory

Short-Term-Memory system enables automatic comprehension of complex linguistic constructs and maintaining conversational context.
